# Data Science Salary Predictor

Predicts Data Science salaries in USD based on job role, experience level, company size, and remote ratio.

## Features
- Predict salary based on your profile
- Compare 4 ML models (Linear Regression, Decision Tree, Random Forest, Gradient Boosting)
- Explore salary trends across experience levels, company sizes and job titles
- Based on 597 real DS job records from 2020–2022

## Tech Stack
Python · Pandas · Scikit-learn · Matplotlib · Streamlit

## Live Demo
https://employee-salary-predict.streamlit.app/

## Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
```
